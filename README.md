# Метеостанция

Приложение для получения и визуализации данных с метеостанции через последовательный порт.

## Использованные инструменты

- Язык C++17
- Фреймворк Qt 5.12.10
- модуль QSerialPort для работы с последовательными портами
- библиотека QCustomPlot для отображения графиков

## Основной функционал

- Подключение к устройству с настройкой соединения.
- Поддержка бинарного протокола и NMEA-формата.
- отобржение показаний датчиков: температура, давление, влажность, CO2, уровень воды, освещенность, шум.
- Отрисовка графиков с возможностью настройки внешнего вида.
- Система уведомлений при выходе показаний за заданные пороги.
- Настраиваемый интерфейс.
